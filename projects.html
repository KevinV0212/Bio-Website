EventTexter

Web application to coordinate events via Twillio text-message API

EventTexter

EventTexter is a Java-based web application that allows users to manage invites to events via text.

It was created using the Vaadin platform (A platform for building web applications with java) and incorporated the Twillio API (an interface for coordinating text messages)

My contribution: At this point in time, I was still inexperienced in software development. Accordingly, I took on smaller roles in the development of this project. I was responsible for researching the Twillio API and connecting it to our web app.

Additionally, I managed the deliverables that we had to submit to the Hackathon.
